🚀 КАК ИСПОЛЬЗОВАТЬ:

1. Скачайте репозиторий с GitHub
2. Запустите COMPILE_WITH_MINHOOK.bat (требуется Visual Studio)
3. Получите version.dll в папке DeftHack_ImGui/
4. Запустите Unturned
5. Инжектируйте version.dll через инжектор
6. Нажмите INSERT для открытия меню
